## Installation

1. Install the development version of `geok` from GitHub:
```R
remotes::install_github("izi-i3i/geok")
```

> Note: geok requires R version > 4.1, otherwise it cannot be installed.

## Usage


**1.** Load package.
```R
library(geok)
```
